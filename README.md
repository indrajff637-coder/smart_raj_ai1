# 📱 SmartRaj AI
## A Modern, Fast & Smart AI Chat App

SmartRaj AI ek free AI app hai jo aapko smart chat, future typing suggestions, 
photo/video sharing aur camera use karne me madad karta hai.  
App ka design **white background + black brain icon** ke saath clean aur minimal rakha gaya hai.

---

## 🚀 Features

### 🤖 Smart AI Chat
- Fast response
- Human-like conversation
- Helpful answers

### ⌨️ Future Typing Suggestions
- Smart next-word prediction
- Fast typing experience

### 📸 Media Support
- Photo bhejo
- Video bhejo
- Camera open karo aur directly send karo

### 🎨 Clean UI
- White themed design
- Black brain logo
- Modern bold “SmartRaj AI” text (right aligned)

## 📥 How to Install

1. APK download kare  
2. Phone me install kare  
3. App open kare → **Splash Screen** sabse pehle dikhegi  
4. Home screen par AI chat + media tools use kar sakte ho
